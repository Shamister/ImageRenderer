Image renderer program (.ppm and .pbm format only)

To load the image, just click the button and direct to the image directory
There 2 kind of images, static and animated.

For static image (not animated image), user can select "Render .ppm image".

"Render Animated Image" is for animated image. The sample images are provided within program folder.
The 1st button can also load animated image file, but it does not animate the image.
